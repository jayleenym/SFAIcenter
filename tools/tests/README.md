# 파이프라인 테스트

## 개요

이 디렉토리는 전체 파이프라인을 테스트하기 위한 Mock 테스트를 포함합니다.

## 테스트 실행 방법

### 1. 의존성 설치

```bash
pip install pandas numpy tqdm openpyxl
```

### 2. Mock 테스트 실행

```bash
python tools/tests/test_pipeline_mock.py
```

## 테스트 내용

- **Mock API 사용**: 실제 API 키를 사용하지 않고 Mock 응답을 반환합니다
- **전체 파이프라인 테스트**: Step 0~4까지의 기본 파이프라인을 테스트합니다
- **임시 디렉토리 사용**: 테스트는 임시 디렉토리에서 실행되며, 실제 데이터를 건드리지 않습니다

## Mock 응답

- **도메인/서브도메인 분류**: 모든 문제를 "경제 > 미시경제학"으로 분류
- **Answer Type 분류**: 모든 문제를 "right"로 분류

## 주의사항

- 실제 API 키는 절대 사용되지 않습니다
- 테스트 후 임시 디렉토리는 자동으로 삭제되지 않습니다 (디버깅용)
- 테스트 데이터는 최소한의 구조만 포함합니다
